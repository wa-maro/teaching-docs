## Web App Setup

### Installation

Install the dependencies:

```bash
$ cd web
$ npm install
```

### Development

To start a local development server, run:

```bash
$ npm run start
```

### Building

To build the project run:

```bash
$ npm run build
```

### Running unit tests

For unit testing, run:

```bash
$ npm run test
```

### Running end-to-end tests

For end-to-end (e2e) testing, run:

```bash
$ ng e2e
```
